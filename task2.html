data1 = data %>% filter(!is.na(loan_amount))

1. Distribution of loan payment

data1 %>%
  ggplot() +
  geom_histogram(aes(x=loan_payment),bins=40, fill = 'pink', color = 'black') +
  labs(title = "Distribution of loan payment") +
  xlab("loan payment") +
  ylab("Frequency")

We see that the loan payment is distributed from 0 to 10000 and is mostly frequent at around 1800.

2. Distribution of loan amount with loan default

data1 %>%
  ggplot() +
  geom_histogram(aes(x=loan_amount,col = loan_default, fill=loan_default),bins=50)+
  labs(title = "Distribution of loan amount with loan default") +
  xlab("loan amount") +
  ylab("Frequency")

We see that non-default loans are much more than default loans.

3. Distribution of loan amount with loan status

data1 %>%
  ggplot() +
  geom_histogram(aes(x=loan_amount,col = loan_status, fill=loan_status),bins=50)+
  labs(title = "Distribution of loan amount with loan status") +
  xlab("loan amount") +
  ylab("Frequency")

We see that current loans are much more than expired loans.

4. Boxplot of loan amount with different loan_term and loan default

data1 %>%
  ggplot() +
  geom_boxplot(aes(x=factor(loan_term),y=loan_amount,fill=loan_default))+
  theme_bw() +
  labs(title = "Boxplot of loan amount with different loan_term and loan default") +
  xlab("loan term") +
  ylab("loan amount")

We see from above box plot that when the loan term is longer, the accounts tend to have more loan amount and are more likely to default.
